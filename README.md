# Hermes Agent

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Python](https://img.shields.io/badge/Python-3.10%2B-green.svg)](https://python.org)

**Hermes Agent** — AI-powered autonomous coding agent. An intelligent CLI assistant that integrates deeply with your development workflow.

## Features

- 🤖 **Autonomous Coding** — Plans, writes, tests and refactors code independently
- 🛠 **Full Toolchain** — Terminal access, file editing, Git/GitHub integration
- 🧠 **Multi-Model** — Supports multiple LLM backends (Claude, GPT-4, DeepSeek, etc.)
- 📦 **Plugin System** — Extensible skill-based architecture
- 🔄 **CI/CD Ready** — GitHub Actions workflows for automated testing and deployment

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Zhang1219318076/hermes-agent.git
cd hermes-agent

# Install dependencies
pip install -r requirements.txt

# Run Hermes
python -m hermes
```

## Requirements

- Python 3.10 or later
- Git 2.30 or later
